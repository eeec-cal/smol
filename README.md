# What it is
`smol` is a fetch scrpit made by Kittyy. It is simple & minimalistic. Plus, it looks very nice.

# How to install it
`smol` only works on Arch-based distributions. To install it, run:
```
yay -S smol
```
Or:
```
paru -S smol
```
